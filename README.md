## 📌 项目简介

本项目为「准爸爸怀孕助手App」的课程设计作品，包含两大部分调研材料：

### 1. 深度需求调研问卷（Formative）
- **文件：** `问卷_深度需求调研.html`
- **用途：** 了解目标用户（准爸爸）的真实痛点、功能期望、知识需求和设计偏好
- **形式：** 线上可填写的H5问卷，共7个模块，预计10-15分钟

### 2. 可用性测试材料包
- **文件：** `准爸爸App_可用性测试材料包_完整版.md`
- **用途：** 预测试/后测试问卷 + 任务场景设计
- **形式：** Markdown格式，可直接打印或线上使用

## 📊 问卷结构（7个模块）

| 模块 | 内容 | 题量 |
|------|------|------|
| 用户画像 | 年龄、职业、孕期阶段、技术熟练度 | 6题 |
| 痛点场景 | 10个真实情境单选题 | 10题 |
| 功能期望 | 12项功能重要性评分（1-5分） | 12题 |
| 知识需求 | 10个知识维度多选题（选3-5个） | 1题 |
| 设计偏好 | 视觉风格、内容形式、付费意愿等 | 6题 |
| 开放问题 | 理想App描述、梦想功能 | 3题 |
| 完成页 | 感谢 + 数据导出 | - |

## 🎯 核心研究发现

### 高优先级痛点
1. **信息缺失** — 不知道产检时该问什么、被问宝宝多大答不上来
2. **角色模糊** — 感觉只是"陪同工具人"，不知道该做什么
3. **应急无助** — 遇到突发情况不知道该问谁、该不该去医院
4. **沟通障碍** — 不知道怎么和妻子、长辈沟通育儿问题

### 高需求功能
- 孕周与宝宝发育状态
- 产检提醒与计划
- 应急处理指南
- 待产包清单
- 知识搜索

### 知识维度偏好（Top 5）
1. 医疗路径（检查流程、指标解读）
2. 风险控制（何时就医）
3. 生活干预（饮食禁忌）
4. 心理与身份适应
5. 育儿预演（临产准备）
